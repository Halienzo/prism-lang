# 叁镜选校系统 (3mir.cc) 深度解构分析报告

> 分析日期: 2026-03-17
> 分析对象: https://3mir.cc/
> 系统全称: 叁镜选校系统 (Smart School Selection)
> 开发公司: 佛山市叁镜教育科技有限公司
> 技术栈: Nuxt.js (Vue) + Vue Query + CNZZ 统计 + GoAdmin 后台

---

## 一、系统整体定位

### 1.1 产品定位
- **B端工具**：面向留学行业**顾问老师/留学中介机构**，非直接面向学生
- **核心价值**：将传统人工查校选校流程从数小时压缩至 **3 分钟**
- **商业模式**：SaaS 付费订阅制（机构账号 + 个人账号）

### 1.2 解决的痛点
| 痛点 | 叁镜方案 |
|------|----------|
| 顾问水平参差不齐 | 标准化院校清单 + 自动匹配 |
| 查找资料耗时长 | 专业关键词检索 + 智能匹配 |
| 方案准确率低 | 严格遵循各校官方 list |
| 案例积累碎片化 | 10万+ 录取案例数据库 |
| 学生管理混乱 | 内置 CRM 系统 |

## 二、系统架构与页面结构

### 2.1 已知路由/页面
| 路由 | 页面名称 | 说明 |
|------|---------|------|
| `/` | 智能选校 (首页) | 主入口，Nuxt.js SPA 动态渲染 |

### 2.2 技术架构
```
前端: Nuxt.js (Vue SSR/CSR 混合)
状态管理: Vue Query (TanStack Query)
API通信: RESTful (staleTime: 5000ms)
数据统计: CNZZ (ID: 1281458735)
后台管理: GoAdmin
AI引擎: ChatGPT-4 Turbo API
部署: Atlassian Pipelines (CI/CD)
协议格式: Protocol Buffers (abproto)
```

## 三、十大功能模块详解

### 模块 1: 专业查询

**覆盖地区 (8个):**

| # | 地区 | 特殊功能 |
|---|------|---------|
| 1 | 英国 (UK) | 支持按学位等级筛选: 一等、二等一、有条件二等二、二等二 |
| 2 | 美国 (USA) | 本科/研究生 |
| 3 | 澳大利亚 (Australia) | 严格遵循院校清单 |
| 4 | 新加坡 (Singapore) | - |
| 5 | 中国香港 (Hong Kong) | - |
| 6 | 中国澳门 (Macau) | - |
| 7 | 中外合办 (Joint Programs) | 严格遵循院校清单 |
| 8 | 爱尔兰 (Ireland) | 严格遵循院校清单 |

**查询方式:**
- 输入中文/英文专业关键词
- 按地区筛选
- 按学位等级筛选 (英国特有)
- 专业标签化分类

### 模块 2: 智能匹配

**输入参数:**
- 本科学校名称 (院校背景)
- 本科均分/GPA
- 意向留学地区
- 意向专业方向

**匹配逻辑:**
- 英国、澳大利亚、爱尔兰、中外合办: **严格遵循院校清单 (List)**
  - 英国各校有自己的中国大学分档 (如利兹 Tier 1A/1B/2/3, 爱丁堡 Band A-D)
  - 根据本科院校所在档位 + 均分自动判定申请资格
- 美国、香港、新加坡、澳门: 基于录取案例数据的概率匹配

**输出结果:**
- 有机会申请的学校和专业列表
- 各专业的录取要求详情
- 匹配度/申请成功概率参考

### 模块 3: 方案生成

- 将匹配到的合适专业添加进**收藏夹**
- 一键生成 **PDF 完整版院校方案**
- 方案自动带上机构 logo
- 内容包含: 专业完整信息、录取要求、语言要求等
- 可直接发送给学生/家长

### 模块 4: 录取案例库

- **数据规模**: 10万+ 条录取案例
- **时间跨度**: 2021-2024年 (21-24年)
- **覆盖范围**: 主流留学申请地区
- **案例字段**:
  - 学生院校背景
  - 录取的学校和专业
  - 其他加分项 (实习/科研/竞赛等)
  - 录取结果 (offer/拒信/候补)

### 模块 5: 院校资讯

- **更新频率**: 每日更新
- **内容类型**:
  - 申请要求变更
  - 语言政策更新
  - 入学政策调整
  - 申请开放/截止通知
  - 新增专业通知
- **营销功能**: 一键生成带机构 logo 的新闻海报

### 模块 6: AI 文书

- **AI 引擎**: ChatGPT-4 Turbo API
- **生成速度**: 3-5 分钟
- **支持文书类型**:
  - 个人简历 (CV/Resume)
  - 个人陈述 (Personal Statement / SOP)
  - 推荐信 (Reference Letter)
- **工作流**: 填写文书素材模板 -> AI 生成初稿 -> 人工修改

### 模块 7: Offer 生成器 / 宣传物料

- **多种 offer 模板**可选
- 导入 offer 信息
- 自动生成带有:
  - 机构 logo
  - 个人微信二维码
  - 学校和专业信息
- **用途**: 朋友圈宣传素材、社交媒体营销

### 模块 8: CRM 客户管理系统

- **学生全流程管理**:
  - 资料收集与储存
  - 申请 DDL 提醒
  - 笔试/面试 DDL 提醒
  - 换无条件 offer DDL 提醒
  - 缴费/签证节点提醒
- **目标**: 不漏掉每一个学生的关键进展节点

### 模块 9: 企业账号管理

- 以**公司为单位**创建账号
- 管理者可查看团队成员的:
  - 匹配记录
  - 案例查看记录
  - 收藏方案记录
- 支持**导出 Excel 表格**数据
- 支持员工绩效和工作量管理

### 模块 10: 自定义专业

- 可自行创建叁镜系统中**未收录**的任意专业
- 适用于新开设专业或小众专业
- 自定义后可纳入匹配和方案生成流程

## 五、设计思路与产品逻辑

### 5.1 核心设计理念
```
输入少量信息 (院校+均分) -> 系统智能匹配 -> 一键输出完整方案
```

### 5.2 用户旅程 (顾问视角)
```
1. 接待学生 -> 获取院校背景/均分/意向
2. 进入系统 -> 输入参数 -> 智能匹配
3. 查看案例 -> 参考历年录取数据
4. 选择专业 -> 添加收藏夹
5. 生成方案 -> 导出 PDF (带机构 logo)
6. 发送给学生/家长
7. 录入 CRM -> 跟踪申请进度
8. AI 生成文书 -> 人工修改 -> 提交申请
9. 收到 offer -> 生成宣传海报
```

### 5.3 数据准确性保障
- 英国/澳洲/爱尔兰: 严格遵循各校官方发布的 **院校清单 (List)**
- 定期更新院校录取政策和要求变化
- 10万+ 真实案例作为匹配参考

## 七、你还没有想到的 - 改进建议

基于对叁镜系统的深度分析，以下是你的棱镜 Prism 系统可以借鉴和补充的方向：

### 7.1 数据维度扩展
- [ ] **GPA/均分维度**: 叁镜的核心匹配逻辑是 "院校背景+均分"，你的系统目前只有语言成绩维度。建议增加 GPA 输入和匹配功能
- [ ] **院校背景分档**: 英国大学有严格的中国院校 List (Tier 分档)，这是选校的关键因素，建议内置主要英校的 List 数据
- [ ] **录取案例库**: 真实录取案例是极有价值的参考数据，考虑增加案例收集和展示功能

### 7.2 缺失的地区
- [ ] **中国香港**: 港八大是热门留学目的地，你的系统目前未覆盖
- [ ] **中国澳门**: 近年申请热度上升
- [ ] **加拿大**: 叁镜也未覆盖，但加拿大是主流留学国之一 (多大/UBC/麦吉尔等)
- [ ] **爱尔兰**: 圣三一等性价比高的选择
- [ ] **中外合办项目**: 如宁波诺丁汉、西交利物浦，是 Plan B 选择
- [ ] **日本/韩国**: 亚洲留学热度持续上升
- [ ] **欧洲大陆**: 荷兰/德国/法国等免学费或低学费目的地

### 7.3 功能层面缺失
- [ ] **方案导出 PDF**: 叁镜可一键生成带 logo 的 PDF 方案，你的系统只能页面浏览
- [ ] **文书辅助**: AI 文书生成是高价值功能，可考虑接入 AI API
- [ ] **申请 DDL 管理**: 你有截止日期日历，但缺少完整的申请流程管理 (资料提交、笔试面试、offer 跟踪等)
- [ ] **院校资讯/动态**: 实时的政策变更推送 (如某校暂停接受新版TOEFL)
- [ ] **录取概率预测**: 基于历史数据给出 "冲刺/稳妥/保底" 更精准的概率
- [ ] **费用估算**: 学费+生活费+签证费的综合预算计算器
- [ ] **签证政策板块**: 各国签证要求、材料清单、PSW/OPT 等工签政策
- [ ] **奖学金信息**: 各校奖学金类型、金额、申请条件

### 7.4 用户体验优化
- [ ] **对比功能增强**: 支持同时对比 3-5 个学校/专业的详细信息
- [ ] **个人档案**: 让学生保存自己的背景信息 (院校+GPA+语言成绩), 一键匹配
- [ ] **分享功能**: 生成可分享的选校方案链接/图片
- [ ] **多语言界面**: 英文/繁体中文版本
- [ ] **申请时间线**: 可视化的申请时间轴 (从选校到入学的完整流程)

### 7.5 数据层面升级
- [ ] **QS/THE/ARWU 排名集成**: 多维度排名对比
- [ ] **就业数据**: 各专业毕业生就业率、起薪、就业去向
- [ ] **学长学姐评价**: 真实就读体验和评分
- [ ] **专业课程设置**: 核心课程列表，帮助理解专业内容
- [ ] **申请材料清单**: 各校各专业所需的具体申请材料列表

### 7.6 你的系统的独特优势 (叁镜没有的)
- **TOEFL 新旧双制度完整支持** — 这是当前市场独有的
- **ETS 官方换算引擎** — 精确的 1-6 制转换
- **集训排课系统** — 叁镜完全没有教学管理功能
- **离线可用 (PWA)** — 叁镜依赖服务器
- **完全免费开源** — 叁镜是付费 SaaS
- **数据质量仪表盘** — 主动监控数据完整性

## 九、结论

叁镜选校系统 (3mir.cc) 是一个成熟的 **B 端留学顾问工具**，核心价值在于"院校背景+GPA -> 智能匹配 -> 方案输出"的闭环流程，加上 10万+ 案例库和 CRM 管理。

你的棱镜 Prism 系统走的是完全不同的路线 — **C 端语言成绩工具 + 教学排课**。两者互补性强，如果融合叁镜的 GPA 匹配逻辑 + 你的语言成绩体系，将形成市场上最完整的选校工具。

**最大的差距不在功能多少，而在数据维度**：叁镜靠"院校背景+GPA"匹配，你靠"语言成绩"匹配。真正的选校决策需要两者结合。建议优先补充 GPA 维度和香港/加拿大地区数据。
